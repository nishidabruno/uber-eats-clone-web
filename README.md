## About The Project
This project is a simplified version of Uber Eats designed for learning and experimentation. It includes core features for building a food delivery app.

Key features:
* Manage Shops: Set up and manage your own virtual restaurant or store. Add menus, items, and details to showcase what you offer.
* Place Orders: Users can browse through available shops, select items, and place orders. A straightforward ordering system simulates the buying experience.
* Map Integration: Visualize shop locations, giving users an intuitive way to see where they are ordering from.

## Getting Started

### Prerequisites

* Node - 18 or above

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:nishidabruno/uber-eats-clone-web.git
   ```
3. Install NPM packages
   ```sh
   pnpm install
   ```
4. Setup environment variables
   ```sh
   # See .env.example
   ```
5. Start the project
   ```sh
   pnpm dev
   ```
